About This Project

This website started as a project to help me find board games in the libraries of Kensigton and Chelsea London and share with friends the amazing titles that are available.

The official catalogue is available at . The data used in this website was extracted from this catalogue with a simple serach by "board game".

While the idea was floating in my ehad for a while, the LibrariesHacked project inspired me to put it to the public test. Give them a visit!

Features

  • Index table: Sort board games by any title, year, etc.
  • Navbar?: Clearly the website is mostly the table for now.

Roadmap

  • BGG integration in index table: An application is ongoing to access the BGG XML API. Once approved expect BGG rank and cathegories.
  • Filtering: Add filtering options to the index table (cathegories, age range) - depends on above...
  • More libraries? Surely more libraries around have board games collections... shoot me a message if you know of any!
  • A map! All fun websites need a map... maybe with the libraries locations?
